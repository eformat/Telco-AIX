## Create your Service & Route Definitions

```bash
kind: Service
apiVersion: v1
metadata:
  name: service-crm
  namespace: tme-aix
spec:
  ports:
    - protocol: TCP
      port: 15000
      targetPort: 15000
  selector:
    statefulset: tme-aix-wb01 <-- This is your RHOAI WEB Name
```

```bash
kind: Route
apiVersion: route.openshift.io/v1
metadata:
  name: route-crm
  namespace: tme-aix
spec:
  path: /
  to:
    kind: Service
    name: service-crm
    weight: 100
  port:
    targetPort: 15000
  wildcardPolicy: None
```
